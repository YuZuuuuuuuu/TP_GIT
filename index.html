Commandes Git Courantes

Initialiser un dépôt Git

Utilisez cette commande pour initialiser un dépôt Git dans un répertoire :

git init

Cloner un dépôt

Pour cloner un dépôt depuis un URL :

git clone <url>

Vérifier l'état du dépôt

Voir les modifications locales et l'état des fichiers :

git status

Ajouter des fichiers

Ajoutez des fichiers pour les préparer au commit :

git add <fichier>

Pour ajouter tous les fichiers :

git add .

Créer un commit

Créez un commit avec un message descriptif :

git commit -m "Message du commit"

Pousser les modifications

Envoyez les modifications locales vers le dépôt distant :

git push

Récupérer les modifications

Synchronisez votre dépôt local avec le dépôt distant :

git pull

Voir l'historique des commits

Consultez l'historique des modifications :

git log

Créer une branche

Pour créer une nouvelle branche :

git branch <nom-de-branche>

Changer de branche

Pour basculer vers une branche existante :

git checkout <nom-de-branche>
